# Installation

## Requirements

- Node.js 18 or later
- npm (comes with Node.js)

## Install globally

```bash
npm install -g promptswap
```

## Verify installation

```bash
promptswap --help
```

You should see the list of available commands.

## First-time setup

After installing, run the setup command to create your account and configure integrations:

```bash
promptswap setup
```

This will:

- Create an account on the promptswap network (or log in to an existing one)
- Install the `/use` skill for Claude Code
- Install the rate-limit recovery hook
- Add shell aliases (`use`, `earn`, `host`)

See [Setup](/docs/setup) for details.
